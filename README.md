# Concert Ticketing Microservices in AWS
REST API Concert Ticketing Microservices Application in AWS to show a distributed architecture using API Gateway, Lambdas and DynamoDB.

#### Requirements
- Docker
- node v22
- aws sam
- npm install -g esbuild
